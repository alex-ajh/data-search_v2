파일 접근 방법
  1. 경로 복사: 검색 결과에서 파일 경로 옆의 버튼을 클릭
  2. 탐색기 열기: Windows 탐색기를 실행 Win + E
  3. 경로 붙여넣기: 주소창에 복사한 경로를 붙여넣기 Ctrl + V
  4. 파일 열기: 해당 위치에서 파일을 확인하고 실행

  • Esc - 검색어 지우기
