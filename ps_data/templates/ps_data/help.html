검색 옵션
검색 유형 검색 방법
2개 이상의 검색어
공백(스페이스바)
예제) 파일명에 "scanner", "보고서"가 있는 경우
파일 확장자 검색
.(파일 확장자)
예제) 파일명에 "scanner"가 있고 파일 확장자가 ppt일 경우
파일 용량으로 검색
size:>(xxMB) 또는 size:<(xxMB)
예제) 파일명에 "scanner"가 있고 파일 용량이 300KB이상이고 5MB이하일 경우
파일 수정 날짜로 검색
s_date:(연월일) 또는 e_date:(연월일)
예제) 파일명에 "scanner"가 있고 파일수정날짜가 2019년05월03일부터 2021년01월03일인 경우
특정 파일 경로에서 검색
dir:(파일 경로)/
예제)CS 산업용 미팅/2021년 폴더안의 파일 검색
